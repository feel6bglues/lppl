## 6. 怎么判断好坏

### 6.1 Template A

通过条件：

- 样本外年化超额收益保持正值
- 样本外最大回撤不明显恶化
- 样本外交易次数和换手率继续低于旧基线

### 6.2 Template B

通过条件：

- `399006.SH` 和 `000852.SH` 的样本外结果都保持正收益或接近正收益
- 回撤可控
- 交易次数不暴涨

### 6.3 失败条件

以下情况直接判失败：

- 样本外结果明显崩掉
- 只剩单个指数成立
- 全量仍然被高换手拖死
