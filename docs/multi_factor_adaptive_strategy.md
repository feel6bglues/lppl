## 策略详细逻辑

### 1. 趋势检测层

**主信号：MA金叉/死叉**
- 短期MA上穿中期MA = 买入信号 (+1分)
- 短期MA下穿中期MA = 卖出信号 (-1分)

**确认信号：多时间框架(HTF)趋势**
- 使用长期MA(120/180/240)判断大趋势
- 价格 > 长期MA = 上升趋势 (+0.5分)
- 价格 < 长期MA = 下降趋势 (-0.5分)

### 2. 波动率过滤层

**ATR波动率比**
- ATR/ATR_MA < 0.95 = 低波动 (+1分，利于入场)
- 0.95 <= ATR/ATR_MA <= 1.05 = 正常 (0分)
- ATR/ATR_MA > 1.15 = 高波动 (-1分，谨慎入场)

**BB宽度市场状态**
- BB_width < 0.05 = 收窄/趋势启动 (+0.5分)
- BB_width > 0.10 = 宽幅/震荡 (-0.5分)

### 3. 信号评分层

**买入条件：**
- 总分 >= 2.0 且 趋势因子 > 0 = 执行买入
- 总分 1.5-2.0 = 观望
- 总分 < 1.5 = 不买入

**卖出条件：**
- 总分 <= -1.5 且 趋势因子 < 0 = 执行卖出
- 总分 -1.5 到 -1.0 = 减仓50%
- 总分 > -1.0 = 持有

### 4. 风险管理层

**ATR动态止损：**
- 止损价 = 入场价 - N × ATR (N=2-3)
- 随ATR调整止损距离

**仓位管理：**
- 低波动环境：满仓(100%)
- 正常波动：标准仓(70%)
- 高波动环境：减仓(50%)

## 通过标准

1. 平均年化超额收益 >= MA+ATR基线 (+1.66%)
2. 平均最大回撤 <= 基线 (-20.20%)
3. 平均交易次数在40-100之间
4. eligible比例 >= 5/7 (71%)
5. 平均Calmar Ratio >= 0.08
