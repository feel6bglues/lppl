## 九、优化建议

### 9.1 参数优化

1. **使用d300_w180_m120配置**：综合收益最高
2. **重点关注accumulation和spring信号**：收益最高
3. **使用B级置信度**：表现最稳定
4. **优先使用fully_aligned多周期信号**：60-cycle验证有效

### 9.2 风险控制

1. **避免markup阶段信号**：收益为负
2. **避免A级置信度**：样本太少且表现差
3. **增加市场环境过滤**：熊市降低仓位
4. **设置止损**：最大回撤控制在20%以内

### 9.3 后续研究方向

1. **增加牛市/熊市分类器**：提高市场环境识别准确性

3. **研究markup阶段负收益原因**：优化阶段转换识别
