# 局限性 & 后续路线

## 7.1 当前局限性

### 方法论局限

| 局限 | 严重性 | 说明 |
|------|--------|------|
| **单一市场** | 中 | 仅A股7指数验证, 未测试港股/美股/日股 |
| **无统计显著性检验** | 中 | 未做Bootstrap/蒙特卡洛, 无法确认Sharpe > 0的概率 |
| **非事件驱动回测** | 中 | 使用收盘价成交, 未模拟簿记撮合 |
| **参数未做交叉验证** | 低 | 网格搜索在全样本上确定最优参数, 未做独立验证集 |

### 策略局限

| 局限 | 说明 |
|------|------|
| **仅做多** | 策略只生成做多信号(0-95%), 未利用做空机制 |
| **指数级别** | 验证在指数级别, 个股表现可能不同 |
| **无杠杆管理** | 未考虑融资融券的杠杆效应 |
| **无动态风控** | 未加入波动率调整止损 |

### 代码质量局限

| 局限 | 说明 |
|------|------|
| **研究原型** | 代码质量停留在"能运行"级别, 非生产级 |
| **重复代码多** | classify_phase_row等函数在3个文件中各有一份 |
| **无测试** | 无单元测试/集成测试 |
| **硬编码参数** | 0.85, 0.50, 0.00 等魔术数字散落代码 |
| **无类型系统** | 参数传递使用dict/基本类型, 无Pydantic校验 |

## 7.2 后续路线

### P0 (1-2周): 生产级回测框架

| 任务 | 说明 | 预计效果 |
|------|------|---------|
| 事件驱动引擎 | 支持VWAP/TWAP滑动成交 | 回测精度±0.3% |
| 限价单模拟 | 簿记撮合而非收盘价成交 | 更真实滑点估算 |
| 多品种支持 | 股票/ETF/期货统一接口 | 扩展性 |

### P1 (2-3周): 统计显著性验证

| 任务 | 方法 | 接受标准 |
|------|------|---------|
| Bootstrap | 收益序列重采样10000次 | Sharpe > 0 概率 > 95% |
| 蒙特卡洛 | 随机信号生成10000次 | 策略Sharpe位于前5% |
| IC测试 | 因子IC/IR检验 | IR > 0.5 |

### P2 (3-4周): 多市场验证

| 市场 | 标的 | 预期适配参数 |
|------|------|------------|
| 港股 | 恒生指数 | MA60/1% 或 MA120/2% |
| 美股 | S&P 500, Nasdaq | MA120/2% (趋势性弱) |
| 日股 | 日经225 | MA60/1% (类似A股) |

### P3 (4-6周): 策略增强

| 增强 | 方法 | 预期增量 |
|------|------|---------|
| 波动率调整 | 高波动时降低仓位 | Calmar +0.3 |
| 动态阈值 | 根据市场波动调整1-3% | Calmar +0.2 |
| 止损机制 | 单日跌幅>3%时强制减仓 | 回撤-5% |
| 杠杆管理 | 低回撤时适度加杠杆 | 收益+3-5% |

### P4 (6-8周): 模拟盘

| 阶段 | 时长 | 目标 |
|------|------|------|
| 模拟盘 | 1个月 | 验证实际执行与回测偏差 < 20% |
| 小资金试盘 | 1个月 | 确认流动性/滑点/冲击成本 |
| 逐步加仓 | 2个月 | 从10%逐步到50%资金 |

## 7.3 预期收益区间 (最终估算)

```
乐观 (P0-P4全部完成):       28-35%年化, Calmar 2.5-3.0
基准 (只完成P0):           25-30%年化, Calmar 2.0-2.5
悲观 (仅当前状态):         15-20%年化, Calmar 1.5-2.0
```

## 7.4 已知风险

| 风险 | 概率 | 影响 | 缓释措施 |
|------|------|------|---------|
| A股市场风格切换 | 中 | 策略失效 | 多市场配置 |
| 印花税/交易规则变化 | 低 | 成本增加 | 动态成本模型 |
| 黑天鹅事件 | 低 | 大幅回撤 | 止损机制 |
| 策略容量 | 低 | 收益稀释 | 指数ETF容量充足 |

## 7.5 最终建议

1. **不要急着实盘**: 当前仍为研究原型, 至少需要P0阶段的工程重构
2. **先做多市场验证**: 如果仅A股有效, 可能是市场特定的幸存者偏差
3. **管理预期**: 计算出的28-31%年化含过拟合偏差, 合理预期实盘在15-20%
4. **关注夏普而非年化**: 1.7-2.0的夏普说明风险调整后收益优秀, 比绝对年化更有意义
