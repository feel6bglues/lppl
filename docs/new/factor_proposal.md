# 超额收益因子可能性分析

> 基于5114只A股 × 12年TDX日线OHLCV数据的可行性研究
> 分析日期: 2026-05-12

---

## 一、已测试因子总结

### 已证伪因子

| 因子 | 年化 | 夏普 | 失败原因 |
|------|------|------|---------|
| MA60/1% 时间序列趋势跟踪 | -3.7% | -0.01 | 交易成本 + 个股反转效应抵消 |
| Wyckoff相位(原始规则) | -3.4% | -0.01 | 方向预测正确但延迟+成本侵蚀 |
| MA60+Wyckoff叠加 | -4.9% | -0.03 | 趋势尾部加杠杆 |
| 双因素确认 | -3.7% | -0.07 | 过度保守 |

### 核心教训

1. **时间序列趋势跟踪在个股层面无效** — 个股波动中反转效应强于趋势
2. **简单信号/单一因子/日频交易** → 交易成本吃掉所有收益
3. **Wyckoff规则描述已发生趋势, 非预测未来** — 滞后性是致命伤

## 二、新因子设计原则

基于5114只个股 × 12年数据的经验:

| 原则 | 原因 |
|------|------|
| **横截面 > 时间序列** | 比较选股而非择时, 多空对冲降波动 |
| **低频 > 高频** | 月频调仓降低交易成本 |
| **多空 > 纯多** | 对冲市场风险, 提取纯alpha |
| **复合 > 单一** | 多个弱信号组合优于单个强信号 |
| **非线性 > 线性** | A股非线性特征显著(涨跌停, 散户行为) |

## 四、最值得优先测试的3个因子

### 1. 短期反转 (预期IR最高)

**逻辑**: A股散户追涨杀跌 → 过去1-2周涨幅最大的股票会在未来1周回调

**参数**:
- 持仓期: 1周 (5交易日)
- 形成期: 2周 (10交易日)
- 做多: 过去2周跌幅前10%
- 做空: 过去2周涨幅前10%
- 调仓频率: 周频

**为什么可能成功**: 
- 与我们已测试的时间序列MA不同 (MA是趋势跟踪, 这是反向)
- A股散户行为偏差极强 → 过度反应→反转
- 周频调仓控制成本

### 2. 涨跌停动量

**逻辑**: A股10%涨跌停机制 → 涨停股票次日高开概率大 (pent-up demand)

**参数**:
- 买入: 今日涨停且封单/成交额比 > 2
- 卖出: 次日开盘
- 做空: 今日跌停, 次日低开

**为什么可能成功**:
- 这是A股市场机制造成的独特套利机会
- 涨停板制度制造了人为的价格发现延迟
- 本质是"昨日无法买入的需求在今天释放"

### 3. 截面多因子等权

**逻辑**: 多个弱信号组合 > 单个强信号

**因子组成**:
1. Past 6M momentum (skip 1M)
2. Past 1W reversal
3. IVOL (past 1M)
4. Amihud illiquidity
5. 涨停动量

**方法**: 每周调仓, 每因子Z-score标准化, 等权合成

## 六、测试建议

优先级排序:
1. **短期反转** (P0, 1-2天实现, 预期IR 0.5-0.8)
2. **涨跌停动量** (P0, 1天实现, A股独特)
3. **截面多因子** (P1, 3天实现, 综合效果)
4. **52周新高** (P1, 1天实现, 经典因子)
