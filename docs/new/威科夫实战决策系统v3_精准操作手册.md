## 第一部分：铁律（违反即出局）

### 铁律1：先定BC，再定方向

**问题根源**：文档C（王府井初版）的灾难性错误——把16.98元的买入高潮(BC)完全漏掉，将下跌途中的震荡误判为积累。

**执行规则**：

```
打开任何一张图表后的第一个动作：

STEP 1：找到最近一次"天量/爆量+长上影线"的高点 → 这就是BC候选
STEP 2：确认BC之后的价格行为：
        ├─ BC后价格持续走低 + 每次反弹 Lower Highs → Distribution/Markdown → 🚫 禁止做多
        ├─ BC后价格在下方区间横盘 + 量能逐步萎缩 → 可能是积累 → ⏸️ 继续观察
        └─ 无法明确判断 → 回到"不知道" → 不交易

⛔ 绝对禁止：在未定位BC之前给出任何方向性判断
```

**自检清单（每次分析必答）**：

| # | 问题 | 答案 |
|---|------|------|
| 1 | 最近的天量/爆量高点在哪里？ | __________元 |
| 2 | 该高点之后的K线是否呈现Lower Highs？ | 是 / 否 |
| 3 | 我判断的"积累区间"是否可能实际上是"分销区间"？ | 是 / 否 / 不确定 |

> **如果第3题答案是"不确定"，默认按分销处理。宁可错过，不可做错。**

### 铁律2："我不知道" = 正确答案

**数据支撑**：

| 文档 | 表态方式 | 得分 | 后果 |
|------|---------|------|------|
| 王府井初版C | "偏多，Bullish Bias" | 2.5 | 若执行亏损-10%+ |
| 润泽初版E | "尚不能断言，高度观望" | 8.2 | 完全避免风险 |

**执行规则**：

```
置信度判定标准（强制执行）：

🟢 A级 — 高置信（可考虑≤20%仓位）
   条件：≥4个独立证据互相印证
   例：三全食品修正版B（周线全多+Spring地量+SOS放量+Phase D结构）

🟡 B级 — 中等置信（仅观察，不操作）
   条件：2-3个证据，无明显矛盾
   例：润泽初版E（80元支撑待验证+方向不确定）

🔴 C级 — 低置信（绝对不操作）
   条件：证据矛盾或不足
   例：王府井初版C实际应归为此类（但当时错判为A级）

⚫ D级 — 无法判断（彻底忽略）
   条件：信息严重不足
   → 直接跳过该标的，不浪费认知资源
```

**关键心态调整**：

> **市场永远在那里。今天不做这个机会，明天还会有别的。"错过"的成本是零；"做错"的成本是你的本金。**

### 铁律3：Spring必须过"冷冻期"

**问题根源**：文档C在3/24以"极高置信度"宣称11.88是Spring并建议建仓，但11.88只是下跌中途站。

**执行规则**：

```
【Spring冷冻期制度】

当你认为发现了Spring信号时：

T日：疑似Spring出现（价格刺穿下边界后拉回）
T+1日：观察 —— 允许观察，禁止建仓
T+2日：观察 —— 允许观察，禁止建仓
T+3日：【解冻】若以下条件全部满足，方可考虑入场：
    ✅ 价格仍保持在Spring低点上方
    ✅ 出现SOS放量阳线
    ✅ 周线级别趋势不矛盾
    ❌ 任一条件不满足 → 继续冷冻或放弃

例外：如果刺穿时量能为"极度萎缩（地量）"
    → 冷冻期可缩短至 T+1日
    → 但仍需 SOS 确认后才可入场
```

**王府井11.88的反面教材**：

```
文档C的错误流程（❌）：
3/23: 11.88出现 → 当日即宣布"Spring成立"
3/24: "极高置信度"、"明日全力建仓"
结果：11.88继续跌破 → 亏损

正确流程（✅）应为：
3/23: 11.88出现 → 标记为"Spring候选"，进入冷冻期
3/24: 冷冻期内，仅观察 → 发现无法快速拉回、继续创新低
3/25: 冷冻期结束 → 条件不满足 → 判定"非Spring"，放弃
→ 完全避免亏损
```

### 铁律4：盈亏比<1:2.5 = 不交易

**这不是建议，这是数学。**

```
【盈亏比速算表】

入场价     止损位      目标位      R:R        操作？
─────────────────────────────────────────────
12.65      11.30      13.00      1:0.26     ❌
12.10      11.30      13.00      1:1.12     ❌
13.00      12.40      14.00      1:1.67     ❌ 接近但不够
11.80      11.30      13.27      1:2.97     ✅ 唯一达标

计算公式：
R:R = |目标价 - 入场价| ÷ |入场价 - 止损价|
要求：R:R ≥ 2.5
```

**为什么必须是2.5？**

| 因素 | 说明 |
|------|------|
| A股T+1 | 次日才能止损，实际损失 > 理论止损 |
| 滑点成本 | 实际成交价偏离预期 |
| 心理压力 | 浮亏时的非理性决策概率上升 |
| 机会成本 | 资金锁定期间的其他机会 |
| **合计** | 需要 ~40% 的安全边际缓冲 → 2.5倍正好覆盖 |

### 每一步的详细检查清单

#### STEP 1：定位BC（耗时：30秒）

```
□ 在图表右侧向左扫描
□ 找到最近一根"量能显著高于周边 + 带上影线"的K线
□ 记录其价格和时间
□ 如果找不到明显的BC → 归入D级，跳过该标的
```

#### STEP 2：判断BC后走势（耗时：1分钟）

```
□ BC后的高点序列：是否逐个降低？（Lower Highs = 分销）
□ BC后的低点序列：是否逐个降低？（Lower Lows = 下降趋势）
□ BC后的量能特征：反弹时缩量？下跌时相对放量？
□ 判定结论：Distribution/Accumulation/不确定
```

#### STEP 3：多周期交叉验证（耗时：2分钟）

```
□ 月线：当前价格 vs MA60 的位置关系？（上方=危险区）
□ 周线：均线排列方向？（全多/全空/混乱）
□ 日线：与周线方向是否一致？
□ 冲突处理原则：长周期压倒短周期
   → 月线空 + 日线多 = 听月线的 = 不做
   → 月线多 + 日线空 = 可以关注日线级别的机会
```

#### STEP 4：寻找Spring（耗时：持续监控）

```
Spring的标准形态（必须全部满足）：
□ 存在一个明确的交易区间下边界
□ 价格短暂刺穿该下边界
□ 当日内或次日快速收回
□ 刺穿时量能：地量（理想）或放量但次日缩量收回（次优）

⚠️ 常见的假Spring陷阱：
  - 下跌途中普通新低（如王府井11.88）
  - 无边界的"自由落体"式低点
  - 刺穿后无法快速拉回（停留超过2天）
```

#### STEP 5：盈亏比计算（耗时：30秒）

```
□ 入场价 = 当前Trigger触发价（不是现在的价格！）
□ 止损价 = 结构性保护位（Spring极点/关键大阳线底部）
□ 目标价 = 上方最近强阻力区（BC区域/前高/缺口）
□ R:R = |目标-入场| ÷ |入场-止损|
□ R:R ≥ 2.5？ → ✅ 否则 ❌
```

## 第四部分：常见陷阱与避坑指南

### 陷阱1：把BC当突破

| 特征 | 真正的突破(Markup启动) | BC(买入高潮) |
|------|----------------------|-------------|
| 量能 | 温和放大或适中 | **天量/爆量**（数倍于均值） |
| K线形态 | 干净的实体阳线 | **长上影线 / 巨量十字星** |
| 后续走势 | 回调幅度浅(<5%)、时间短 | **回调深(>10%)、持续时间长** |
| 均线配合 | 多周期共振向上 | 仅短期向上，中长期仍向下或混乱 |
| 三全食品案例 | — | 13.27元就是典型BC |

**口诀**：**天量+长上影 = BC = 不是突破 = 别追！**

---

### 陷阱2：把下跌中途站当Spring

| 特征 | 真正的Spring | 假Spring（下跌站） |
|------|-------------|------------------|
| 前置条件 | 存在明确的积累区间 | 无区间，纯下跌 |
| 刺穿深度 | 轻微刺穿下边界 | 大幅创新低 |
| 拉回速度 | **当日/次日快速拉回** | 缓慢或不拉回 |
| 量能 | 地量（理想） | 任意 |
| 后续行为 | SOS放量阳线确认 | 继续阴跌或弱势震荡 |
| 王府井案例 | — | 11.88元就是典型假Spring |

**口诀**：**没有区间就没有Spring。下跌途中每一个低点都可能是更低点的起点。**

---

### 陷阱3：Confirmation Bias（确认偏误）

**症状自查**：

```
你在追踪过程中是否出现过以下想法？

☐ "再坚持一下，马上就突破了"
☐ "这次缩量意味着变盘在即"
☐ "主力在洗盘，别被洗出去"
☐ "已经跌了这么多，不可能再跌了"

如果有 ≥2 个 ☐ → 你已陷入Confirmation Bias
→ 立即停止当前分析
→ 重新从STEP 1开始（重新定位BC）
→ 用全新的眼光审视图表
```

**对抗方法**：

```
【强制反事实练习】
每次做出"看多"判断后，强制自己写一段"看空论证"：

"如果我是错的，市场会表现出：
1. ________________________
2. ________________________
3. ________________________

当前图表是否匹配以上反事实预测？
□ 是 → 推翻原判断
□ 否 → 维持原判断，但降低置信度一级"
```

---

### 陷阱4：FOMO（错失恐惧症）

**症状**：看到别人赚钱或某个标的大涨，忍不住想追。

**解药**：

```
【FOMO冷却清单】

追涨前必须回答：
1. 这个机会我是在它启动前就分析的，还是看到涨了才想起来的？
   → 后者 = FOMO，禁止操作

2. 我现在知道它的BC位置吗？
   → 不知道 = 违反铁律1，禁止操作

3. 如果我在当前价买入，我的止损位在哪里？盈亏比够2.5吗？
   → 不够 = 违反铁律4，禁止操作

4. 这个标的在我的观察列表里吗？我追踪了几天？
   → 没有/不到3天 = 信息不足，禁止操作

以上4题任一否定 → 今天不交易。明天再说。
```

| 数字 | 来源 | 含义 | 使用场景 |
|------|------|------|---------|
