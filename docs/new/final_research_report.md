## 二、核心发现

### 发现1: OHLCV数据中的常见因子在A股均不产生alpha

12个候选因子全部失败:
- **时间序列类**: MA趋势跟踪, Wyckoff相位, 双因素确认
- **截面类**: 动量, 反转, 涨跌停, 多因子组合
- **结果**: 无一能稳定战胜买入持有(等权)

### 发现2: Wyckoff相位方向预测正确但无法转化为交易收益

相位收益验证 (n=5000+):
- markup次日: **+0.82%** / 57.2%胜率 ✅
- markdown次日: **-0.53%** / 41.2%胜率 ✅

方向预测正确, 但交易成本 + 信号延迟 + 低频覆盖 → 实际收益 -3.4%

### 发现3: A股个股的反转效应强于趋势效应

- 短期反转(买跌卖涨)获得 **-10.5%** 年化 = 实际是"追涨杀跌"更好
- 这意味A股个股表现出**短期动量**(过去2周涨的继续涨)
- 这与成熟市场不同: 中国A股的散户行为导致趋势延续而非反转

### 发现4: 截面动量也失败

经典12-1月动量策略在A股:
- 年化 -3.9%, 夏普 -0.61
- 为什么? A股高换手率+强反转=动量利润被交易成本完全吃掉
- 即使在月度调仓下, 双边0.25%的成本已经足够侵蚀动量收益

### 发现5: 涨跌停动量是唯一正夏普的策略

- 夏普 +1.80 (所有因子中最高)
- 但年化 -1.9% (仍是负收益)
- 原因: 绝大多数时间仓位为0(极少涨停), 夏普被大量零收益日拉高

## 三、原因分析

### 为什么OHLCV因子在A股不工作?

| 原因 | 说明 |
|------|------|
| **散户主导 + 高换手** | 散户交易占比80%+, 年换手率200-400% — 有效市场假说不成立, 但噪声交易者的对手方风险也高 |
| **交易成本高** | A股印花税千1 + 佣金万2.5 + 滑点 = 单边0.15-0.25%, 是美股的3-5倍 |
| **涨跌停制度** | 10%限制扭曲价格发现, 导致因子信号延迟 |
| **T+1限制** | 当日不能卖出, 加仓后必须隔日才能调整, 增加风险 |
| **IPO制度** | 新股上市连续涨停, 导致IPO后一段时间无交易机会 |

## 四、最终结论

### 评分汇总

| 维度 | 评分 | 说明 |
|------|------|------|
| 已测试因子数量 | **12** | 时间序列5 + 截面4 + 组合3 |
| 有效因子数量 | **0** | 无一能稳定战胜买入持有 |
| 方向预测正确 | **2/12** | Wyckoff markup/markdown, 但交易不足以转化 |
| 数据覆盖面 | **5000+只 × 12年** | 行业内最大样本之一 |
| 结论可靠度 | **高** | 所有回测修正了前瞻偏差 + 扣除交易成本 |

### 一句话结论

> **仅靠A股日线OHLCV数据, 在扣除交易成本和修正回测偏差后, 无法稳定获得显著超额收益。**

### 后续建议

1. **如果需要继续**: 获取Level2/基本面/资金流数据
2. **如果追求实用**: 指数级别的趋势跟踪 + Wyckoff相位(如最初7指数验证的10-16%年化)
3. **如果需要部署**: 重构当前代码为生产级回测框架, 接入更多数据源
