# 研究概览与关键发现

## 1.1 研究背景

本研究的起点是一个复杂的多因子组合系统 (FCE v1, 401行代码), 试图通过加权评分融合 LPPL 泡沫检测、Wyckoff 相位分析、MTF 多周期对齐、置信度分级等因子生成交易信号。经过 7 轮迭代验证, 最终发现一个仅 3 行规则的纯制度策略稳健战胜买入持有, 且 Wyckoff 相位在此基础上提供+3.3%年化增量。

## 1.2 研究路线

```
FCE v1 (复杂多因子, 401行)
  ↓ 实盘验证: 7/7指数负收益 (-1.4%年化)
发现1: 纯制度策略 > 复杂多因子
  ↓ 大范围验证: 7/7指数有效
P1网格搜索: MA60+1% >> MA120+3%
  ↓ 210组合×7指数
P2 Wyckoff增量: +3.3%年化 (有成本下)
  ↓ 原始analyzer.py规则提取
交易成本模型: -1.9%/年侵蚀
  ↓ 含佣金+印花税+滑点
Walk-forward验证: 3折独立时间段
  ↓ 训练→测试分离
最终: MA60趋势 + Wyckoff微调 = 28-31%年化/Calmar 2.8
```

## 1.3 关键数据汇总

| 指标 | 买入持有 | 纯制度(有成本) | 制度+Wyckoff(有成本) |
|------|---------|--------------|-------------------|
| 平均年化 | 5.8% | 25.6% | 28.9% |
| 平均夏普 | 0.36 | 1.70 | 1.80 |
| 平均回撤 | -58.9% | -18.6% | -16.2% |
| 平均Calmar | 0.10 | 1.98 | 2.49 |
| 超额收益 | — | +19.8% | +23.1% |

## 1.4 核心发现

| 发现 | 证据 | 置信度 |
|------|------|--------|
| 纯制度策略 7/7指数有效 | 平均超额+19.8%年化 | 高 |
| MA60优于MA120 | 网格搜索210组合, MA60在7/7最优 | 高 |
| Wyckoff相位有独立增量 | 有成本下+3.3%年化, 7/7改善Calmar | 中 |
| 交易成本可控 | -1.9%/年, 占收益7% | 高 |
| Walk-forward通过 | 3折样本外全部有效 | 高 |
| 复杂多因子系统失败 | FCE v1全部负收益, v2仅1.9% | 高 |
