## 二、LPPL泡沫过滤算法详解

### 2.1 LPPL模型简介

LPPL（Log-Periodic Power Law）模型是一种用于检测金融泡沫和崩盘的数学模型。该模型基于以下假设：

1. **对数周期性振荡**：价格在接近临界点时呈现对数周期性振荡
2. **幂律增长**：价格以幂律形式增长
3. **临界点**：存在一个临界时间点tc，表示泡沫破裂的时间

LPPL模型的数学表达式为：

```
ln(p(t)) = a + b(tc-t)^m + c(tc-t)^m * cos(w*ln(tc-t) + phi)
```

其中：
- `p(t)`：t时刻的价格
- `tc`：临界时间点（泡沫破裂时间）
- `m`：幂律指数（0 < m < 1）
- `w`：对数周期振荡频率（6 < w < 13）
- `a, b, c, phi`：拟合参数

### 2.2 泡沫检测算法

本系统采用的泡沫检测算法基于以下技术指标：

#### 2.2.1 价格偏离度检测

```python
# 计算价格相对于均线的偏离度
price_deviation = (close - ma60) / ma60

# 泡沫条件：价格偏离度超过20%
if price_deviation > 0.2:
    is_bubble = True
```

**原理**：当价格大幅偏离均线时，表明市场可能过热，存在泡沫风险。

#### 2.2.2 波动率检测

```python
# 计算20日波动率
returns = close.pct_change()
volatility = returns.rolling(20).std()

# 泡沫条件：日波动率超过3%
if volatility > 0.03:
    is_bubble = True
```

**原理**：高波动率通常伴随市场情绪极度亢奋，是泡沫的典型特征。

#### 2.2.3 相对位置检测

```python
# 计算价格在120日区间内的相对位置
high_120 = close.rolling(120).max()
low_120 = close.rolling(120).min()
relative_position = (close - low_120) / (high_120 - low_120)

# 泡沫条件：相对位置超过95%
if relative_position > 0.95:
    is_bubble = True
```

**原理**：当价格处于历史高位附近时，上涨空间有限，下跌风险增大。

### 2.3 泡沫阶段合并算法

检测到的泡沫阶段可能存在重叠，需要进行合并：

```python
def merge_bubble_periods(bubble_periods):
    """合并重叠的泡沫阶段"""
    if not bubble_periods:
        return []
    
    merged_periods = []
    current_start, current_end = bubble_periods[0]
    
    for start, end in bubble_periods[1:]:
        if start <= current_end:
            # 重叠，合并
            current_end = max(current_end, end)
        else:
            # 不重叠，保存当前段，开始新段
            merged_periods.append((current_start, current_end))
            current_start, current_end = start, end
    
    merged_periods.append((current_start, current_end))
    return merged_periods
```

## 三、空仓时段分析

### 3.1 空仓时段定义

空仓时段是指市场处于泡沫状态或高风险状态，投资者应避免入场或减仓的时期。

### 3.2 空仓时段识别方法

#### 方法一：LPPL泡沫检测

通过LPPL模型检测市场泡沫，当检测到以下信号时标记为空仓时段：

1. **价格偏离度 > 20%**：价格大幅偏离均线
2. **波动率 > 3%**：市场情绪极度亢奋
3. **相对位置 > 95%**：价格处于历史高位

#### 方法二：技术指标组合

结合多种技术指标进行综合判断：

```python
def is_empty_position_period(date, index_data):
    """判断是否为空仓时段"""
    row = index_data[index_data['date'] == date]
    
    # 条件1：价格偏离均线
    if row['close'] > row['ma60'] * 1.2:
        return True
    
    # 条件2：波动率过高
    if row['volatility'] > 0.03:
        return True
    
    # 条件3：相对位置过高
    if row['relative_position'] > 0.95:
        return True
    
    return False
```

def generate_cycle_specs(bubble_periods, seed=42):
    """生成10个随机测试时点"""
    random.seed(seed)
    specs = []
    
    # 尝试生成10个不在泡沫阶段的日期
    attempts = 0
    max_attempts = 1000
    
    while len(specs) < 10 and attempts < max_attempts:
        attempts += 1
        
        # 随机选择年份（2012-2025）
        year = random.randint(2012, 2025)

        month = random.randint(3, 11)

        date_str = f"{year}-{month:02d}-{day:02d}"
        
        # 检查是否在泡沫阶段
        if not is_in_bubble_period(date_str, bubble_periods):

                cycle_id=len(specs) + 1,
                year=year,
                as_of_date=date_str,
