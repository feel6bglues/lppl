## 四、评估标准

### 4.1 持有时间

| 指标 | 目标值 | 评估标准 |
|------|--------|----------|
| 平均持仓周期 (大盘) | ≥ 8周 | 满足目标 |
| 平均持仓周期 (高波动) | ≥ 4周 | 满足目标 |
| 最长持仓周期 | ≥ 26周 | 优秀 |
| 最短持仓周期 | ≥ 2周 | 合理 |

### 4.2 收益与风险

| 指标 | 目标值 | 评估标准 |
|------|--------|----------|
| 年化超额收益 | ≥ 0% | 满足目标 |
| 最大回撤 | ≤ -15% | 满足目标 |
| Calmar比率 | ≥ 0.3 | 良好 |
| 胜率 | ≥ 50% | 良好 |

### 4.3 交易效率

| 指标 | 目标值 | 评估标准 |
|------|--------|----------|
| 年化换手率 | ≤ 8% | 满足目标 |
| 交易次数 | 明显低于高频版本 | 满足目标 |
| whipsaw_rate | ≤ 0.3 | 良好 |
| 逃顶成功率 | ≥ 70% | 良好 |

### 4.4 暴跌窗口表现

**重点监控窗口**:
- 2015年股灾
- 2018年贸易战
- 2020年疫情
- 2024年调整

**评估标准**:
- 是否提前切到 CASH 状态
- 最大回撤是否低于基线
- 逃顶后重新入场时机是否合理

## 七、风险控制

### 7.1 潜在风险

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 参数过拟合 | 样本外表现差 | 严格控制参数范围 |
| 逃顶信号过晚 | 损失过大 | 优化逃顶触发条件 |
| 重新入场过早 | 反复交易 | 增加冷却期和确认条件 |
| 数据质量问题 | 测试结果不可靠 | 数据完整性检查 |

### 7.2 失败条件

如果出现以下情况，本次调优视为失败：
- 持仓周期未达到目标
- 逃顶效果不明显
- 样本外表现显著退化
- 换手率高于现有策略
- 全量测试年化超额为负
