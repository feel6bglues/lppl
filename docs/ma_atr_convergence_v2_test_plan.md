# MA均线收敛 + ATR波动率策略V2优化测试计划

日期：2026-03-31

## 优化背景

V1策略（布林带收口+ATR分位数为主信号）测试结果：
- 平均年化超额收益：-0.36%（未通过）
- 平均交易次数：20（过于保守）
- eligible比例：3/7 (43%)

**问题分析：**
1. 布林带收口条件过于严格，导致交易机会过少
2. ATR分位数作为主信号，过滤了太多有效信号
3. 缺乏明确的趋势跟踪能力

## V2优化方向

### 核心改变

| 项目 | V1 | V2 |
|------|----|----|
| 主交易信号 | 布林带收口+ATR分位数 | MA金叉/死叉 |
| 布林带作用 | 主信号（收口判断） | 辅助过滤（放宽阈值） |
| ATR作用 | 主信号（分位数） | 辅助评分（波动率确认） |

### V2交易逻辑

**买入条件（需满足所有主条件 + 至少1个辅助条件）：**

主条件：
1. MA金叉（短期均线上穿中期均线）
2. 价格在长期趋势线之上（regime_filter）

辅助条件（二选一）：
1. ATR低波（atr_ratio < atr_low_threshold）
2. 布林带相对收窄（bb_width < bb宽阈值）

**卖出条件（满足任一主条件）：**

主条件：
1. MA死叉（短期均线下穿中期均线）
2. 跌破长期趋势线 + 回撤超阈值

辅助加成：
- ATR高波时加速减仓

## 通过标准

1. 平均年化超额收益 >= MA+ATR基线 (+1.66%)
2. 平均最大回撤不差于基线 (-20.20%)
3. 平均交易次数在30-80之间（合理范围）
4. eligible比例 >= 5/7 (71%)
