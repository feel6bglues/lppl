## 一、LPPL 参数分析

### 1.1 参数含义说明

LPPL 模型参数数组格式为 `[tc, m, w, w2, a, b, c]`，其中：
- `tc`：临界时间（泡沫破裂预测时间点）
- `m`：幂律指数（0.1-0.9 区间，越低表示趋势越弱）
- `w`：对数周期振荡频率
- `w2`：阻尼项
- `a`：对数价格基准值
- `b`：幂律振幅（负值表示下跌趋势）
- `c`：对数周期振荡振幅
- `rmse`：模型拟合误差（越低越好）

## 三、LPPL + Wyckoff 结合调优方案

### 3.1 核心思路

LPPL 提供市场趋势强度判断（m 值）和拟合可靠性判断（RMSE），Wyckoff 提供市场周期阶段判断和信号生成。两者结合的逻辑是：

```
LPPL 判断"趋势是否可能反转" + Wyckoff 判断"当前处于什么阶段" = 综合信号
```

### 3.2 m 值与 Wyckoff 阶段映射关系

| LPPL m 值范围 | 含义 | 对应 Wyckoff 阶段 | 操作方向 |
|--------------|------|------------------|----------|
| 0.1 - 0.3 | 趋势极弱，可能见底 | Accumulation 末期 / Markdown 末期 | **买入/加仓** |
| 0.3 - 0.6 | 趋势不明朗 | Unknown / 震荡 | **观望** |
| 0.6 - 0.9 | 趋势较强 | Markup / Distribution | **持有/减仓** |
| 0.9 (边界) | 趋势极强，需警惕反转 | Distribution 末期 | **减仓/清仓** |

### 3.3 分场景调优方案

#### 场景 A：基准为负（大盘下跌/熊市）

**适用数据集：** wyckoff_excess_returns, wyckoff_6cycle_best_dates

**推荐组合：**

| 条件 | 具体值 | 数据支撑 |
|------|--------|----------|
| LPPL m 值 | < 0.35 | 长期参数中 m 值 0.2-0.3 对应底部区域 |
| LPPL RMSE | < 0.03 | 短期 RMSE 范围 0.013-0.022 |
| Wyckoff 阶段 | Accumulation 或 Spring | 超额收益 +15.64%, 胜率 75% |
| Wyckoff 信号 | Accumulation 信号 | 超额收益 +22.69%, 胜率 88.4% |
| Wyckoff 置信度 | B 级 | 超额收益 +8.49%, 胜率 60.9% |
| Wyckoff 窗口 | 300天 | 整体超额 +6.02% vs 600天 -1.66% |

**预期效果：**
- 超额收益：+15% ~ +23%
- 胜率：65% ~ 80%
- 适用周期：2013, 2021 等熊转牛阶段

#### 场景 B：基准为正（大盘上涨/牛市）

**适用数据集：** wyckoff_8cycle_excess_returns

**推荐组合：**

| 条件 | 具体值 | 数据支撑 |
|------|--------|----------|
| LPPL m 值 | > 0.5（确认趋势） | 中期参数中 m=0.9 对应上涨趋势 |
| LPPL RMSE | < 0.025 | 高拟合度保证信号可靠 |
| Wyckoff 阶段 | Markdown（相对强势股） | 超额收益 +0.38%, 是唯一正收益阶段 |
| Wyckoff 信号 | SOS Candidate | 超额收益 +5.14%, 在正基准下最强 |
| Wyckoff 置信度 | B 级 | 超额收益 +1.77%, 相对最优 |
| **方向选择** | **空仓观望** | 超额 -0.52%, 优于持有观察(-5.55%) |

**预期效果：**
- 超额收益：+0% ~ +5%（牛市中很难跑赢大盘）
- 胜率：40% ~ 50%
- **建议：牛市中降低主动选股比例，以指数 ETF 为主**

#### 场景 C：震荡市

**推荐组合：**

| 条件 | 具体值 |
